# Ghani Africa - African Digital Marketplace

  Africa's leading digital marketplace connecting businesses, traders, manufacturers, and consumers across all 54 African countries.

  ## Features
  - Secure escrow payments with Stripe integration
  - 50+ African currencies with automatic conversion
  - WhatsApp integration for direct seller communication
  - Bulk/tiered wholesale pricing
  - Sample orders with special pricing
  - Group buying / cooperative purchasing
  - AfCFTA trade compliance tools & duty calculator
  - Mobile money payment infrastructure (M-Pesa, MTN, Airtel, Orange)
  - Multi-language support (11 languages including Arabic RTL)
  - Photo reviews system
  - Supplier factory profiles
  - Enhanced offline PWA with service worker caching
  - Push notifications via Expo mobile app
  - Real-time order tracking and logistics
  - 25+ pickup points across major African cities

  ## Tech Stack
  - **Frontend**: React 18, TypeScript, Tailwind CSS, Vite
  - **Backend**: Express.js, TypeScript
  - **Database**: PostgreSQL with Drizzle ORM
  - **Payments**: Stripe + Mobile Money
  - **Email**: SendGrid
  - **AI**: GPT-4 Vision for image search
  - **Mobile**: Expo + React Native WebView (see mobile/ directory)

  ## Getting Started
  ```bash
  npm install
  npm run dev
  ```
